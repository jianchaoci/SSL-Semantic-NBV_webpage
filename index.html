SSL-Semantic-NBV Workflow

An overview of the self-supervised learning process within a single NBV iteration. Each iteration involves eight steps: (i) predict IG for candidate views via a neural network using the current reconstruction and viewpoint history; (ii) select the next viewpoint based on predicted IG; (iii) move the camera and capture new data; (iv) segment specific plant parts from the new observation based on the task; (v) calculate ground-truth IG using the current reconstruction and new observation; (vi) store training samples with the current reconstruction and view history as inputs and ground-truth IG as supervision; (vii) train the network using stored samples; (viii) update reconstruction with the new observation.
